WichtelApp

Benutzeranleitung

WichtelApp ist eine App zum schreiben und lesen von Wichteltexten, welche auf einem Wichteltag gespeichert werden.

Scrollen Sie bitte durch die folgenden Seiten um mehr zu erfahren

Wichteltexte

Wichteltexte sind kurze Textnachrichten mit einer länge von maximal 90 Zeichen.

Wichteltag

Wichteltags sind speziell für diese App formatierte NFC-Tags (Near Field Communication), auf welchen Wichteltexte gespeichert werden können.

Umgang mit NFC-Tags

Der sichere Umgang mit NFC-Tags garantiert eine reibungslose Datenübertragung zwischen NFC fähigem Smartphone und NFC-Tag.

Um einen Wichteltag zu nutzten müssen sie diesen berühren: Berühren Sie den Wichteltag mit dem oberen Bereich der Rückseite Ihres Smartphones. Achten Sie darauf, dass das Gerät mit einer eindeutigen Bewegung dicht an den Wichteltag gebracht wird.

Durch kurzes Vibrieren signalisiert Ihnen WichtelApp, dass die Datenübertragung abgeschlossen wurde. Nach Abschluss der Datenübertragung können Sie das Gerät wieder vom Wichteltag entfernen. Achten Sie auch beim Entfernen des Gerätes auf einen deutlichen Bewegungsablauf. Zitternde, ruckartige oder langsame Bewegungen können zu unerwünschten Ergebnissen führen.

Auslesen eines Wichteltags

Halten Sie bitte einen Wichteltag bereit.

Um die Wichteltexte eines Wichteltags auszulesen klicken Sie im Hauptmenu auf den Menupunkt
[ Scanne Wichteltag ] und folgen Sie den Bildschirmausgaben.

Es kann jeweils nur ein Wichteltag auf einmal gescannt werden! Wenn Sie also einen anderen Wichteltag scannen wollen, gelangen Sie über den runden Button in der oberen linken Ecke des Displays (<) wieder zurück in das Hauptmenu,
von wo aus sie erneut via
[ Scanne Wichteltag ] einen Suchvorgang starten können.

Wichteleditor

Halten Sie bitte einen Wichteltag bereit.

Um einen Wichteltext auf einem Wichteltag abzulegen, muss dieser zuerst im Wichteleditor verfasst werden. Den Wichteleditor erreichen Sie im Hauptmenu über den Menupunkt [ Schreibe Wichteltext ].
Im Textfeld des Wichteleditors können Sie Ihre persönliche Wichtelnachricht verfassen.

Bitte verwenden Sie in Ihren Wichteltexten keine Sonderzeichen wie Geschweifte-, Eckige-Klammern oder Semikolon.

Die Anzahl verwendbaren Zeichen ist auf 90 beschränkt. Über den Button [ link hinzufügen ] können Sie Ihrem Wichteltext eine Hyperlink hinzufügen, dieser wird jeweils am Ende des bereits verfassten Textes angehängt.
Bitte beachten Sie auch beim Einfügen eines Links das lediglich 90 Zeichen für die Nachricht verwendet werden können.

Wenn Sie mit Ihrem Wichteltext zufrieden sind, können Sie auf [ Suche und beschreibe NFC-Tag ] klicken. Folgen Sie den Hinweisen auf dem Bildschirm und Ihre Nachricht wird erfolgreich auf den Wichteltag übertragen und ersetzt automatisch den ältesten sich auf dem Wichteltag befindlichen Wichteltext.
